# Apache Doris (apache-doris)
Apache Doris is a high-performance, real-time analytical database based on MPP architecture. It provides sub-second queries on large-scale data with a MySQL-compatible interface and supports both batch and streaming data ingestion.

## APIs

### Apache Doris
Doris provides a MySQL-compatible protocol for SQL queries, a REST API for cluster management, Stream Load API for real-time data ingestion, Routine Load for Kafka integration, and JDBC/ODBC drivers.
